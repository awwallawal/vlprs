Section 2

Detailed Cost Breakdown

2.1 Software Design, Development & Deployment — ₦6,900,000

Full-stack system design and development covering 14 functional modules and 82 functional requirements, delivered across 6 implementation phases over 28 weeks:

  • 4-tier financial computation engine with precision arithmetic for all loan types
  • Immutable transaction ledger with three-layer integrity enforcement
  • 5 role-specific user interfaces (Accountant General, Deputy AG, Department Admin, MDA Officers, System Admin)
  • Executive dashboard with real-time portfolio metrics optimised for mobile access
  • MDA monthly submission workflow with automated validation and comparison
  • Auto-Stop Certificate generation system with dual notifications
  • Exception management, automated alerts, and reporting engine
  • Retirement pipeline forecasting and gratuity receivable tracking
  • PDF report generation with Oyo State branding
  • Public-facing website for scheme information and transparency
  • Secure authentication, role-based access control, and comprehensive audit logging
  • Automated deployment pipeline for reliable, repeatable releases

2.2 Data Migration & MDA Onboarding — ₦1,500,000

Systematic migration of legacy data from 63 MDA spreadsheets into the VLPRS system:

  • Column mapping and data transformation for up to 63 unique spreadsheet formats
  • Variance categorisation (Clean, Minor, Significant, Structural, Anomalous)
  • Non-punitive side-by-side comparison for each MDA to review and confirm their data
  • Baseline establishment — an agreed-upon verified starting point per MDA
  • Pilot onboarding (3–5 MDAs) followed by phased full deployment

2.4 Staff Training & Capacity Building — ₦850,000

  • 2-day hands-on training workshop covering all user roles
  • Printed training materials and role-specific user guides for each participant
  • Laminated quick-reference cards for daily operations
  • Workshop logistics: refreshments, catering, training supplies
  • Post-workshop video guides for ongoing reference and new staff onboarding

2.5 Hardware Procurement — ₦457,000

  • Desktop workstation for the Car Loan Department — the primary operational terminal for day-to-day system administration
  • Specification: suitable for web-based application use, standard office productivity, and report generation
  • Includes monitor, keyboard, mouse, UPS (uninterruptible power supply), and basic accessories

2.6 Project Documentation & User Manuals — ₦500,000

  • System Administrator Guide — comprehensive technical reference for ongoing management
  • Role-specific User Manuals — tailored handbooks for each user type (Executive, Dept Admin, MDA Officer)
  • Operational Procedures Manual — standard processes for monthly cycles, exceptions, reporting
  • Data Dictionary and System Architecture documentation — for institutional knowledge preservation

2.7 Post-Deployment Support (90 Days) — ₦500,000

  • Priority resolution of reported issues and defects
  • Assistance with MDA onboarding questions during the stabilisation period
  • System performance monitoring and optimisation
  • Minor adjustments based on user feedback during initial live operations

Section 5

Value Analysis

5.1 Investment Context

The Vehicle Loan Scheme represents over ₦2.6 billion in active financial obligations managed on behalf of 3,100+ civil servants across 63 MDAs. The ₦12 million investment in VLPRS represents less than 0.5% of the portfolio it is designed to govern.

5.2 Quantifiable Returns

Area Current Cost / Impact With VLPRS
Report Generation 2–3 working days per cycle (manual compilation from 63 sources) Under 10 seconds, any time, from any device
Calculation Accuracy ~30% discrepancy rate in sampled records 100% computational accuracy — single verified engine
Over-Deduction Prevention Unquantified losses — multiple incidents per year, administrative cost of investigation + refund processing Eliminated — Auto-Stop Certificate at zero balance
Fund Visibility No real-time view of available fund, outstanding obligations, or recovery projections Real-time dashboard — total exposure, fund available, recovery rate
MDA Officer Workload 17-column spreadsheet with manual calculations 8-field submission — 60% workload reduction
Audit Readiness No systematic audit trail — figures untraceable Every transaction permanent, timestamped, attributed — fully auditable
Retirement Recovery Reactive — discovered after the fact Proactive — 3/6/12-month forecasting with gratuity receivable tracking

Table 5: Value Comparison — Current State vs. VLPRS

5.3 The Cost of Inaction

Every month the current manual process continues:

  • Calculation errors continue to compound across 3,100+ loans undetected
  • Workers risk over-deduction with no automated safeguard
  • The Accountant General cannot answer scheme status queries without multi-day manual compilation
  • Retirement obligations remain invisible until after the fact
  • The revolving fund operates without visibility into its own health
  • Every figure produced is indefensible under audit scrutiny

The question is not whether the State can afford this investment. The question is whether the State can afford to continue administering a ₦2.6 billion portfolio using spreadsheets and paper files — and to bear the operational, financial, and reputational consequences of doing so.

Section 6

Terms & Conditions

6.1 Proposal Validity

This financial proposal is valid for 90 days from the date of submission (February 2026). Pricing is subject to review beyond this period due to potential changes in vendor costs and exchange rates.

6.2 Scope

This proposal covers the complete design, development, deployment, data migration, training, hardware procurement, documentation, and 90-day post-deployment support for VLPRS as described in Volume 1 (Technical Proposal, Ref: CKL/VLPRS/TP/2026/001).

6.3 Client Obligations

The Office of the Accountant General will provide:

  • Timely access to existing MDA spreadsheet data for migration
  • A designated project liaison from the Car Loan Department
  • Availability of staff for the 2-day training workshop
  • Reasonable cooperation of MDA reporting officers during the onboarding process
  • Access to a suitable location for hardware installation in the Car Loan Department

6.4 Intellectual Property

Upon full payment, the Office of the Accountant General retains full rights to use the VLPRS system for its intended purpose within the Oyo State Government. Core Knowledge Limited retains ownership of the underlying software architecture and codebase.

6.5 Confidentiality

All data processed by VLPRS — including beneficiary records, financial transactions, and MDA submissions — is treated as confidential government information. Core Knowledge Limited commits to maintaining the confidentiality and security of all data handled during and after the project.
