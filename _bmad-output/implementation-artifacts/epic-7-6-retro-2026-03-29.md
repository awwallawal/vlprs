   - 2026-03-21: ESM `.js` extension crash-loop (7.0b) — 2 same-day hotfixes

   - 2026-03-25: No deploy version verification — health check didn't confirm BUILD_SHA
   - 2026-03-29: PDF export broken — `tsup` doesn't bundle binary assets, Dockerfile missing COPY
2. **"Extend, don't fork" violations** — 23 raw `fetch()` calls accumulated because `apiClient` didn't support FormData. Bypass comments disguised as documentation
3. **Migration computation model flaw** — system reverse-engineers rate from MDA's already-wrong Excel data instead of computing from scheme formula (13.33% ÷ 60)
4. **No pre-acceptance inspection flow** — "Establish Baseline" is accept-or-nothing, no detail view or correction capability
5. **Pace vs polish tension** — 17 stories in 10 days, but deployment pipeline needs investment

## Key Insights

1. **UAT catches what 2,500 tests and code review don't** — Awwal's manual calculation (scheme formula) exposed a computation model flaw invisible to automated testing
2. **Domain expertise is irreplaceable** — the three-vector comparison model (Scheme Expected vs Reverse Engineered vs MDA Declared) only emerged from product owner cross-referencing real data against scheme rules
3. **Wrong data cascades everywhere** — accepting a migration record with impossible outstanding balance (₦671,979 > total loan ₦479,985) creates a negative baseline entry that inflates every downstream metric, report, and PDF
4. **The code review process remains the primary quality gate for code** — but deployment topology has no equivalent gate
5. **Group boundaries are natural retrospective points** — Groups 1+2 complete delivers a shippable core product
6. **Combined retro + live UAT is powerful** — surfaced 10 actionable findings that wouldn't emerge from either alone

### Finding Detail: Three-Vector Computation Model (#4, #5)

Demonstrated with ALATISE BOSEDE SUSAINAH (BIR, Principal ₦450,000, 30-month tenure):

| Vector | Monthly Interest | Total Interest | Total Loan |
|--------|-----------------|---------------|------------|
| **Scheme Expected** (P×13.33%÷60) | ₦999.75 | ₦29,992.50 | **₦479,992.50** |
| **Reverse Engineered** (6.663% from Excel) | ₦999.45 | ₦29,983.50 | ₦479,983.50 |
| **MDA Declared** (raw Excel values) | ₦999.50 | ₦29,985.00 | ₦479,985.00 |

All three disagree. Scheme Expected is authoritative per domain rules.

### Finding Detail: Cascade Impact (#6)

```
Declared Outstanding ₦671,979 (WRONG — exceeds total loan ₦479,985)
  → Baseline entry: ₦479,985 - ₦671,979 = -₦191,994 (NEGATIVE)
  → Balance computation: totalLoan - (-₦191,994) = totalLoan + ₦191,994
  → Inflates: Total Outstanding Receivables, MDA Health Score, Attention Items,
     Recovery Projections, Weekly AG Report, all E6 PDF reports
```
